# BookReader

Simple book reader and editor for books of own format.

This was a project for C++ Programming class in the university.

## Features

- open book from file
- create new book, import text from text file or load text files as chapters
- save book to file
- information about the book: title, author, annotation, date, genre
- chapters
- text search, go to page number
- bookmarks, comments, references
- images
- auto-scroll with time interval
- book library
- settings: font color and style, line interval and text alignment
